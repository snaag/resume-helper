# resume-helper

## 사용 방법
1. components/WriteList.tsx 에서 list 에 각 항목의 `제목 (title)` 과 `최대 길이 (maxLength)`, `부가 설명 (description)` 을 작성
2. `npm run dev` 로 실행 (꼭 dev 여야 하진 않음) 

## 실행 방법
```shell
npm run dev
```
